## 监听对象的操作

比如说从对象中获取数据\给对象某个属性赋值\删除对象属性

```javascript
const obj = { name: "why" };
obj.name;
obj.name = "kobe"; // 监听到赋值的操作 -> 执行一些其他的事情（）
delete obj.name;
```

- 我们先来看一个需求：有一个对象，我们希望监听这个独享中的属性被设置或获取的过程
  - 通过我们前面所学的知识，能不能做到这一点呢？
  - 其实是可以的，我们可以通过之前的属性描述符中的存储属性描述符（Object.defineProperty）来做到。
